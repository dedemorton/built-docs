Fleet and Elastic Agent 7.15.0edit

Review important information about the Fleet and Elastic Agent 7.15.0 releases.

Known issuesedit

Upgrade Elastic Agents to use Osquery Manager integration

Details
You must upgrade your Elastic Agents to the latest version to use the Osquery Manager integration.

Impact
To upgrade, refer to Upgrade Elastic Agents.

Enhancementsedit

Fleet
  • Adds package policy upgrade API #103017
  • Updates Package Policy UI to support upgrading package policies #107171
  • Removes subseconds from event.ingested #104044

Bug fixesedit

Fleet
  • Fixes Fleet settings and HostInput error handling #109418
  • Fixes Agent policy search to support simple text filters #107306
Elastic Agent
  • Adds validation for certificate flags to ensure they are absolute paths #27779
  • Migrates state on upgrade #27825
