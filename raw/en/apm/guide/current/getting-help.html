Get helpedit

Start by searching the APM Server discussion forum for your issue. If you can’t find a resolution, open a new issue or add a comment to an existing one. Make sure you provide the following information, and we’ll help you troubleshoot the problem:

  • APM Server version
  • Operating System
  • Configuration
  • Any supporting information, such as debugging output, that will help us diagnose your problem. See Debug for more details.

If you’re sure you found a bug, you can open a ticket on GitHub. Note, however, that we close GitHub issues containing questions or requests for help if they don’t indicate the presence of a bug.
